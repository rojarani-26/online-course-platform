# 📚 Online Course Platform – SQL Project

## 📌 Project Overview
This project demonstrates the design and analysis of a **relational database for an Online Course Platform** using SQL Server.  
The database stores information about **courses, students, and enrollments**, and SQL queries are used to analyze learning patterns and course performance.

---

## 🎯 Objectives
- Design a **relational database**
- Create **primary and foreign key relationships**
- Manage course enrollment data
- Perform **SQL queries for data analysis**

---

## 🗂️ Database Tables

| Table | Description |
|------|-------------|
| **Courses** | Stores course details such as course name, category, and price |
| **Students** | Stores student information |
| **Enrollments** | Tracks which student enrolled in which course |

## 📊 Key Insights
- The platform contains **15 registered students**.
- **Python for Data Analysis** is the most popular course with **4 enrollments**.
- Data-related courses such as **SQL, Python, and Power BI** attract the most students.
- **Python for Data Analysis generated the highest revenue ($316)**.
- Many students enroll in **multiple courses**, indicating continuous learning behavior.
